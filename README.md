# Yolo8n--Multi-object-detection
YOLOv8n (Nano) is the smallest and fastest model variant in the Ultralytics YOLOv8 family. 
- Input: 5 minute traffic video (9000 frames)
- Resolution: 1920x1080 at 30 FPS
- Detection Speed: ~15-30ms per frame on GPU
- Total Processing Time: ~2-3 minutes with Google Colab GPU

Key Learnings: 
* YOLO model architecture and inference pipeline
* OpenCV video I/O and frame processing
* Data structures for managing detections
* Computer vision workflows and best practices
* Filtering, visualization, and analytics
